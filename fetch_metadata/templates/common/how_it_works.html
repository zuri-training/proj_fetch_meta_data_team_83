Extracting metadata from your first Files?

Let's help you figure out how it works

1

Upload files of your choice from selected personal library, computer or Google Drive ranging from any of the specified file formats indicated above without exceeding a size limit of 250mb by clicking the Browse Button on the Dashboard Page

2

Once uploaded to the website, one can proceed to click on the EXTRACT button close to it so that the system analyzes the file to generate metadata of the uploaded file

3

Once analysis is complete, the website displays the metadata of the uploaded file through an overlay screen that pops up on the screen. Metadata informations like File Size, File Type, etc.

4

Below the displayed metadata of the uploaded files, are two options. Basically, SHARE OR EXPORT. The Share button allows you to either Share the link to the displayed metadata page through Social Channels or Embedd the website in a link.

5

The Export button allows you to export the displayed metadata through COPY through a clipboard or DOWNLOAD the displayed metadata through a file to be downloaded.
