Extract Metadatas from your Files

Extract metadata recorded behind your files, ranging from files size and modification history to the software tools used to create them. Available to image files, csv, pdf, json, etc.

How it Works

Upload files ranging from images, csv, pdf and json, etc. by clicking the Browse button

Click on the Extract button to generate the metadatas embedded in the selected file.

Once generated, you are allow to Copy the extracted metadatas to a clipboard

MetaData from Images

Images contain exif data that can give you useful information about the picture. Information such as shutter speed and focal length are stored inside an image. Likewise, you can find out where the photo was taken by looking at the location information.

An online exif data reader can show you all these hidden information.

MetaData from Files

Documents can contain metadata too. They include information such as file size and date of creation, but also information about the author of a document and the software used to create it.

An exif viewer such as metadata2go.com shows you all you need to know about your text document.
